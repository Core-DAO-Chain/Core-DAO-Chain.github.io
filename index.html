What is Core DAO?

Core site overview describes a Bitcoin-aligned, EVM-compatible layer-one blockchain built around Satoshi Plus consensus, BTC staking, CORE staking, and a growing BTCfi ecosystem.

Core DAO Chain Bitcoin staking yield

Core DAO Chain positions itself around self-custodial Bitcoin staking and Bitcoin DeFi. Its Bitcoin staking design uses Bitcoin timelocks so BTC stays under the user's control while participating in Core's validator election process. Rewards are paid in CORE, and live rates can change with protocol conditions, validator performance, and dual-staking choices. The self-custodial Bitcoin staking FAQ explains the custody model, but users should treat displayed yield as variable rather than guaranteed and avoid relying on promotional ranges as fixed returns.

Core DAO Chain gas fees and RPC

Core is EVM-compatible, so wallets and developers can use familiar Ethereum-style tooling such as MetaMask, Solidity, and block explorers. CORE is the native token used for gas and staking. Users adding the network should rely on Core Mainnet configuration for Chain ID 1116, RPC details, currency symbol, and explorer links, then verify transactions on Core Scan. Actual gas costs still depend on network activity, contract complexity, and the wallet's fee settings.

CORE token utility and supply

CORE is the native utility and governance token of the Core blockchain. It pays transaction fees, participates in staking and governance, and is used in Core's dual-staking design to boost Bitcoin staking rewards. The CoinMarketCap CORE profile describes CORE as the network utility and governance token with a maximum supply of 2.1 billion coins. That makes token utility broader than simple gas, but users should evaluate validator economics, emissions, staking demand, and Core DeFi usage rather than looking only at spot price.

Core DAO dual staking explained

Core dual staking combines self-custodial Bitcoin staking with CORE token staking. The basic idea is that a user timelocks BTC on Bitcoin and stakes CORE in the Core ecosystem, with higher CORE-to-BTC alignment potentially improving reward tiers. This is not the same as wrapping BTC or depositing it with a lending desk; an Investopedia custody overview is a useful baseline for the difference between self-custody and third-party custody. Users should still evaluate lock duration, reward token volatility, validator selection, and the opportunity cost of holding CORE.

Core DAO FAQ

What is Core DAO Chain?

Core DAO Chain is a Bitcoin-aligned, EVM-compatible blockchain focused on Bitcoin staking and Bitcoin DeFi. CoinGecko's Core categories list it among smart contract platform and Bitcoin sidechain assets, while Core uses Satoshi Plus consensus with CORE staking, Bitcoin miner delegation, and Bitcoin timelock participation.

Does Core staking require giving up Bitcoin custody?

Core describes its Bitcoin staking as self-custodial. BTC is timelocked on the Bitcoin network using native Bitcoin scripting rather than transferred to a third-party staking pool, and the Bitcoin transaction scripts guide explains how scripts set spending conditions. Users still need to understand lock duration, reward rules, and validator selection.

What is Satoshi Plus consensus?

Satoshi Plus is Core's consensus design that combines delegated Bitcoin mining power, CORE staking, and Bitcoin holder participation. The CoinMarketCap Satoshi Plus summary describes the mechanism as combining Bitcoin proof-of-work and delegated proof-of-stake concepts.

What is CORE used for?

CORE is used for gas fees, staking, governance, and dual-staking reward alignment in the Core ecosystem. The Core network currency symbol listing names CORE as the Core Mainnet currency symbol, so users need it for network transaction fees as well as staking-related activity.

Can I use MetaMask on Core DAO Chain?

Yes, Core is EVM-compatible, so MetaMask and other EVM wallets can connect when configured with the correct network settings. MetaMask network management documentation explains detecting, switching, and adding EVM networks; after connecting, verify balances and transactions on Core Scan.

Is Core DAO the same as Bitcoin Layer 2?

Core is Bitcoin-aligned and uses Bitcoin-related participation in its consensus, but users should read its architecture rather than assuming it is identical to every Bitcoin Layer 2 label. Stacks Documentation's model overview is a useful contrast because Stacks has its own Bitcoin-linked technical model, while Core is EVM-compatible with Satoshi Plus mechanics and its own CORE token.
