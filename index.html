What is Core DAO?

Core is a Bitcoin-aligned, EVM-compatible layer-one blockchain built around Satoshi Plus consensus, BTC staking, CORE staking, and a growing BTCfi ecosystem.

Before you explore

Bitcoin-aligned design

Satoshi Plus connects Bitcoin miner participation, CORE staking, and BTC staking with an EVM-compatible execution layer.

Official network details

Core Mainnet is documented with Chain ID 1116, CORE as the currency symbol, official RPC, explorer, and staking site in the network configuration.

Core DAO FAQ

What is core dao?

core dao refers to Core, a Bitcoin-aligned, EVM-compatible layer-one blockchain for BTCfi, staking, and smart-contract applications.

Is core dao compatible with Ethereum wallets?

Yes. Core Mainnet is EVM-compatible, and the official docs list Chain ID 1116, CORE as the currency symbol, RPC details, and explorer links.

Where can I explore core dao transactions?

Use Core Scan, the official explorer for blocks, transactions, addresses, and token contracts on Core Mainnet.

Does core dao support Bitcoin staking?

Core supports self-custodial Bitcoin staking through its official staking route, with BTC timelocking used as part of network participation.

What token pays gas on core dao?

CORE is the native currency symbol used for Core Mainnet transaction fees and network activity.
